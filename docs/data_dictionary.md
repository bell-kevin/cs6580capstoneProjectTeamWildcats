# Data Dictionary

This dictionary documents the **joined hourly modeling dataset** produced by `src/data_cleaning.py`.

| Column | Type | Description |
| --- | --- | --- |
| `date_hour` | datetime | Hourly timestamp key used to join traffic and weather records. |
| `traffic_count_total` | float | Total vehicle count across all lanes for that hour. |
| `lane_count` | int | Number of lanes with reported counts in that hour. |
| `temp_f` | float | Mean air temperature (°F) across weather samples in that hour. |
| `dewpoint_f` | float | Mean dew point (°F) in that hour. |
| `humidity_pct` | float | Mean relative humidity percentage in that hour. |
| `altimeter_in` | float | Mean altimeter pressure (inHg) in that hour. |
| `solar_w_m2` | float | Mean solar radiation in W/m² in that hour. |
| `voltage_v` | float | Mean station voltage in volts in that hour. |
| `wind_speed_mph` | float | Mean wind speed in mph in that hour. |
| `gust_mph` | float | Mean wind gust in mph in that hour. |
| `wind_direction_deg` | float | Mean wind direction in degrees in that hour. |
| `snow_depth_in` | float | Mean snow depth in inches in that hour. |
| `precip_1hr_in` | float | Mean precipitation accumulation in inches for the prior hour. |
| `snow_interval_in` | float | Mean snowfall accumulation over the source station interval (inches). |
| `weather_samples_per_hour` | float | Count of weather observations aggregated into that hourly row. |
| `date` | date string | Calendar date extracted from `date_hour` (`YYYY-MM-DD`). |
| `observed_date` | date string | Federal holiday observed date matched from holiday calendar. |
| `holiday_name` | string | Holiday name for matching dates (empty string when not a holiday). |
| `observed_shifted` | string | Holiday observed-day shift indicator from source file. |
| `is_federal_holiday` | bool | `True` when `date` is a U.S. federal observed holiday. |
| `is_weekend` | bool | `True` when the date is Saturday or Sunday. |
| `hour` | int | Hour of day (0–23). |
| `day_of_week` | string | Day name (e.g., Monday, Tuesday). |

## Notes

- The dataset is generated by reading from immutable files in `data/raw` and writing output to `data/processed`.
- Weather fields are hourly means of sub-hourly station readings.
- Traffic rows are lane-normalized and summed to produce an hourly total
